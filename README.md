# My-2048-game
Gamenya itu 2048 cara main game nya itu adalah pertama kamu mulai akna ada 16 tile yang akan spawn random number yaitu 2 dan 4 dan kamu harus combine terus sampai dapat 2048 kamu bisa gerakin tile yang ada nomor dengan W/A/S/D jadi contohnya kamu pencet d akan ke kanan tapi semua tile yang ada nomor akan pergi ke kanan bukan cuman satu dan akan sama dengan W yaitu ke atas dan S yaitu ke bawah dan A yaitu ke kiri
